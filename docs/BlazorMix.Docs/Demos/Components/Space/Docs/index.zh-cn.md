# Space 间距

元素排列中保持相同的宽度。

## 何时使用

适用于多个元素按照水平或垂直方向保持相同的间距。

### CSS 变量

| 属性                  | 说明     | 默认值                |
| --------------------- | -------- | --------------------- |
| --space-gap               | 间距大小              |   `var(--gap)` (8px)    |
| --space-gap-vertical      | 水平方向的间距大小     |   `var(--space-gap)`    |
| --space-gap-horizontal    | 垂直方向的间距大小     |   `var(--space-gap)`    |
