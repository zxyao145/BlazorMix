# NoticeBar 通告栏

展示一组消息通知。

## 何时使用

适用于当前页面内信息的通知，是一种较醒目的页面内通知方式。

### CSS 变量

| 属性               | 说明           | 默认值    |
| ------------------ | -------------- | --------- |
| --background-color | 背景色         | `#ababab` |
| --border-color     | 边框颜色       | `#999999` |
| --font-size        | 文字字号       | `15px`    |
| --height           | 整体高度       | `40px`    |
| --icon-font-size   | 左侧图标的字号 | `18px`    |
| --text-color       | 文字颜色       | `#ffffff` |
