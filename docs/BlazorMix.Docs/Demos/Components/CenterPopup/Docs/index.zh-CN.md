# CenterPopup 中间弹出层

从屏幕**中间**滑出或弹出一块自定义内容区。

## 何时使用

适用于展示弹窗、信息提示、选择输入、切换等内容，支持多个弹出层叠加展示。

### CSS 变量


| 属性               | 说明             | 默认值                                           |
| ------------------ | --------        | ------------------------------------------------ |
| --mask-z-index     | 元素的 `z-index` | 1000                      |
| --popup-bg-color   | 弹出层的背景颜色 | #fff                      |
| --popup-min-width   | 弹出层的最小宽度 | 280px                      |
| --popup-max-width   | 弹出层的最大宽度 | 75vw                      |
